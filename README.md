# SimpleHarvest
Allows players to right click crops and easily harvest them

Supports the following crops
- Beetroot 
- Carrot
- Cocoa
- Netherwart
- Potato
- Wheat

The permissions are
- simpleharvest.* (default)
- simpleharvest.beetroot
- simpleharvest.carrot
- simpleharvest.cocoa
- simpleharvest.netherwart
- simpleharvest.potato
- simpleharvest.wheat
